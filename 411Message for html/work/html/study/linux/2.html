设置中文

终端输入

dpkg-reconfigure locales
权限不足时在命令前添加 sudo
选择
[]zh_CN.UTF-8 UTF-8
使用方向键选择 回车确认
安装完成后重启(命令:reboot)

安装中文输入法

权限不足时在命令前添加 sudo
apt-get install fcitx
apt-get install fcitx-googlepinyin
安装后重启

系统更新

打开终端输入

apt-get update //检查更新
apt-get upgrade //更新文件
apt-get clean //清理缓存文件
权限不足时在命令前添加 sudo
