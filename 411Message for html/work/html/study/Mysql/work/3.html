数据库语句:mysql -h服务器地址(本地主机localhost) -u用户名 -p密码

MySQL配置文件:默认C:\ProgramData\MySQL\MySQL Server 5.7\my.ini

免密登录:修改my.ini 添加:skip-grant-tables 修改后重启SQL服务

创建数据库 CREATE DATABASE [IF NOT EXISITS];
删除数据库 DROP DATABASE;
查看数据库 SHOW DATABASE;
使用数据库 USE DATABASE;

创建message数据库
create database message;
删除数据库
drop database message;
