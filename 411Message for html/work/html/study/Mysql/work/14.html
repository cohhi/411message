数据库备份与恢复 (dump版)

备份数据库:

mysqldump -hlocalhost -uroot -p 数据库名 数据表名 > 备份位置


数据表还原

mysql -hlocalhost -uroot -p 数据库名 < 数据表位置


今日作业

创建tb_work数据表

create table tb_worker(

id INT(11)PRIMARY KEY AUTO_INCREMENT,

name VARCHAR(30)NOT NUll,

sex ENUM('w','m') DEFAULT 'm',

hobby SET('football','basketball','volleyball'),

score FLOAT(3,1) UNSIGNED,

moblie char(11) UNIQUE NOT NULL,

intro char(4) NULL,

entry_time DATE

)ENGINE=MyISAM,charset utf8;

修改数据表

修改name值为小吴的记录 将hobby改为football

update tb_worker SET hobby='football' where name='小吴';


将id值小于4的员工score+0.5

update tb_worker SET score=score+0.5 where id<4


执行以下语句删除sex值为 'w'的员工
