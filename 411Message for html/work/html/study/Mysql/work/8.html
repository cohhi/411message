where子句中的条件判断符
   =     等于
   >     大于
   <     小于
   >=    大于等于
   <=     小于等于
   !=     <> 不相等

关键字 and or
区别 ：
and 代替&& 符合所有的条件
or 代替|| 所有条件中有一个符合条件

关键字 in
字段等于指定内容中的任意一个值就会返回

关键字null
空值is null
非空值 is not null

通配符: "%" "_"
% 匹配任意长度
_ 用法与%类似只能匹配一个字符

between and
需要设置两个参数，起始值和结束值
between 参数1 and 参数2

作业:

代码:

update tb_worker set id=4 where name="小赵" or name="冯晓曼"
